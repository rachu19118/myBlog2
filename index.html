My First Console Project

Visual Studio is a great place to code. You first get into the place by going to the home page and finding a blue logo that says Visual Studio Code, and then you create a coding folder. After that, you open the folder onto Visual Studio. Finally, get the HTML file by clicking the first icon next to the folder name, and then typing index.html in the folder

The console screen is useful to help learn Javascript because it is very complicated. We can't memorize everything in Javascript, so we can show what we know on the Console screen to make sure everything is correct. Then we can make sure that there are not any mistakes made.
